This project is a web-based alternative to SIGAA (Integrated System for Academic Activities Management), designed to provide a simpler, more intuitive, and visually organized way to access information about professors. It addresses common issues found in traditional academic systems, such as scattered information and poor usability, by presenting data in a clean card-based interface with photos, real-time search functionality, and planned features like favorites for personalization. Developed using HTML, CSS, and JavaScript, the system focuses on improving user experience through responsiveness, interactivity, and clear information structure, making it easier for students to quickly find and explore relevant academic information.
