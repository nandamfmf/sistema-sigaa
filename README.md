🎓 Alternative SIGAA System
📌 Description:
This project consists of the development of a web-based system as an alternative to SIGAA (Integrated System for Academic Activities Management), focusing on improving the visualization and access to professors' information.
The application provides a more modern, organized, and intuitive interface, aiming to reduce common issues found in traditional academic systems, such as information decentralization and poor usability.

🎯 Objective:
Provide a simpler and more efficient way to:
View professors
Search for instructors quickly
Organize information clearly

🚀 Features:
Display professors in card format with photos
Real-time search functionality
Favorites system (in development)
Responsive interface (works on different screen sizes)

Technologies Used:
HTML5
CSS3
JavaScript (Vanilla JS)
